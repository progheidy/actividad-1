CUADRO COMPARATIVO ENTRE HTML 5, CSS, JavaScript

Lenguaje ¿Qué es? Ventajas Desventajas Sistema Operativo
HTML 5 Utilizado para la creación de páginas web. HTML significa "hypertext mark-up language", que significa, "lenguaje para el formato de documentos de hipertexto", es el componente más básico de la web, ya que define el significado y la estructura del contenido web.
  • Texto presentado de forma estructurada y agradable.
  • Sencillo que permite describir hipertextos.
  • Archivos pequeños.
  • Lo admiten todos los exploradores.
  • Lenguaje de fácil aprendizaje.
  • No necesita de grandes conocimientos cuando se cuenta con un editor de páginas web.
    		•
  • Lenguaje estático.
  • Las etiquetas son muy limitadas.
  • Lento para ejecutar archivos pesados.
  • Guarda muchas etiquetas que pueden convertirse en "basura" y dificultan la corrección.
  • El diseño es más lento.
  • Las páginas pueden variar su aspecto dependiendo del navegador (es un lenguaje interpretado).
    		•  Se puede ver en todos los sistemas operativos pero la interpretación varía según el navegador que este utilizando.
    CSS Las siglas CSS (Cascading Style Sheets) significan "Hojas de estilo en cascada" y parten de un concepto simple pero muy potente: aplicar estilos (colores, formas, márgenes, etc...). Básicamente es el que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.
  • Implica menos código y probabilidad de errores.
  • Mayor gama de posibilidades de edición para hacer que tu sitio luzca tal como quieras.
  • Los diseñadores web necesitan utilizar pocas líneas de programación para cada página, lo que mejora la velocidad del sitio.
  • Ayuda a formar cambios espontáneos y consistentes.
  • Tiene el poder de reposicionarse. Nos ayuda a determinar los cambios dentro de la posición de los elemntos web que se encuentran en la página.
    		•
  • Existe escasez de seguridad.
  • Mayor dificultad al identificar los elementos afectados por distintas clases de estilo.
  • Después de realizar los cambios, debemos confirmar la compatibilidad si aparecen. El cambio similar afecta a todos los navegadores.
  • Compatibilidad con el navegador (se admiten algunas hojas de estilo y otras no).
  • Ciertas propiedades, como posición o visibilidad de los elementos, pueden provocar que una parte del contenido de la página resulte inaccesible.
    		•  Para todos los sistemas operativos. Algo que suele ocurrir mucho, es que el diseñador elija una fuente que no suele ser estándar para sus diseños.
    JavaScript Es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. (Está bien, no todo, pero es sorprendente lo que puedes lograr con unas pocas líneas de código JavaScript).
  • Los script tienen capacidades limitadas por razones de seguridad.
  • Es multiplataforma, puede ser ejecutado de manera híbrida en cualquier sistema operativo móvil.
  • Lenguaje de scripting seguro y fiable.
  • Tiende a ser muy rápido porque a menudo se ejecuta en el navegador.
  • Los desarrolladores pueden extender la funcionalidad de las páginas web mediante fragmentos de JavaScript.
    		•
  • Código visible por cualquier usuario.
  • Tiende a introducir gran cantidad de fragmentos de código en los sitios web.
  • No es compatible en todos los navegadores de manera uniforme.
  • Sus scripts lo hacen vulnerable.
  • El código debe descargarse completamente.
  • Puede desactivarse por los visitantes.
  • Es ejecutado en el client-side, bugs y descuidos pueden ser explotados algunas veces para malos propósitos.
    		•  Se puede ver en todos los sistemas operativos pero la interpretación varia según el navegador que esté utilizando.
